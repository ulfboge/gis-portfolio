# GIS Portfolio (MkDocs)

En enkel MkDocs-sajt med dummy-projekt för GIS/NVI/våtmarker.

## Bygga lokalt

1. (Valfritt) Skapa och aktivera virtuell miljö
2. Installera beroenden:

```bash
pip install -r requirements.txt
```

3. Starta dev-server:

```bash
mkdocs serve
```

4. Bygg statisk sajt:

```bash
mkdocs build
```
