# Om mig

Välkommen till min GIS-portfölj! Jag är en GIS-konsult med fokus på fjärranalys, naturvärdesinventering och ekologisk analys.

## Kompetensområden

### 🗺️ Geodata & GIS
- QGIS, GRASS GIS, ArcGIS Pro
- Geodataförvaltning och harmonisering
- Rumslig analys och modellering
- HMK-kompatibla leveranser

### 🛰️ Fjärranalys
- Sentinel-1/2 (SAR + optisk)
- Tidsserieanalys och förändringsdetektion
- Vegetationsindex (NDVI, NDWI, NDMI)
- Google Earth Engine, Python (rasterio, xarray)

### 🌿 Naturvärdesinventering (NVI)
- Fototolkning och fältinventering
- Klassning enligt SIS-standard (SS 199000:2014)
- Artdata och biotopkartering
- MKB-underlag och påverkansbedömning

### 📊 Ekologisk analys
- Spridningsanalys (least-cost, graph-theory)
- Konnektivitetsmodellering (Circuitscape, NetworkX)
- Habitatfragmentering
- Ekosystemtjänster

### 💻 Teknisk stack
- **Språk:** Python, R, JavaScript (GEE)
- **Ramverk:** GeoPandas, rasterio, NetworkX, igraph, gdistance
- **Databaser:** PostgreSQL/PostGIS, GeoPackage
- **Rapportering:** MkDocs, LaTeX, Markdown

## Projekt i denna portfölj

Projekten på denna sajt är **dummy-exempel** som illustrerar typiska uppdrag inom svensk GIS-konsultbransch. De är baserade på verkliga upphandlingar men innehåller fiktiva data och resultat.

**Flaggskepp:**
- [**VåtmarkWatch**](projects/nv-vatmarkwatch.md) – Sentinel-baserad våtmarksmonitering
- [**KommunNVI-Suite**](projects/ulricehamn-kommunnvi-suite.md) – NVI + spridningsanalys + MKB-underlag

## Arbetsmetodik

Jag arbetar efter följande principer:

1. **Reproducerbarhet:** Skript och modeller dokumenteras för återanvändning
2. **Kvalitetssäkring:** HMK, intern granskning, metadatadokumentation
3. **Öppen källkod:** Föredrar QGIS, Python, R och öppna dataformat
4. **Kommunikation:** Tydliga rapporter och kartor anpassade för beställare
