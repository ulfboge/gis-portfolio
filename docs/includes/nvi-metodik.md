# NVI-metodik (översikt)

- Klassning: 1–4 enligt SIS-standarden
- Underlag: orto, NNH, fastighets- och planunderlag
- Arbetsflöde: fototolkning → fält (vid behov) → digitalisering → QA
- Leverabler: GPKG (geometri + attribut), kartbilagor, rapport
- HMK: fält för kvalitet, källa, datum, metod, granskare
